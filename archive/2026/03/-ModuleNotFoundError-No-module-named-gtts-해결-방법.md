<h2>문제</h2>
<p>파이썬 스크립트 실행 시에 아래와 같은 에러가 발생함</p>
<table border="1" style="border-collapse: collapse; width: 100%;">

</tr>
</tbody>
</table>
<p>&nbsp;</p>
<h2>해결 방법</h2>
<p>gtts 패키지를 설치</p>
<pre class="python" id="code_1664207892100"><code>$ sudo pip3 install gtts</code></pre>
<p>&nbsp;</p>
<h2>만약 "pip3 명령이 없습니다"라는 에러가 발생할 경우</h2>
<p>pip3 패키지가 설치되지 않은 문제이므로 아래의 명령으로 python3-pip 패키지 설치</p>
<pre class="python" id="code_1664207977236"><code>$ sudo apt-get install python3-pip</code></pre>
